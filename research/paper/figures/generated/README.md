# Generated vector figures

This directory is reserved for reviewed, manifest-derived PDF figures that do
not fit the in-source TikZ/PGFPlots templates. Rasterized plots are not accepted
for the final paper. Any file added here must be generated from validated raw
evidence, use black-and-white encodings that remain legible without color, and
be included explicitly from `main.tex` or a section file.

The arXiv archive target removes this Markdown file.
